## 材质包各文件说明

- 【主材质】：材质包主体文件
- 【模型拓展包】：通过将“绿宝石”物品命名来获得相应模型，添加了400个额外的模型。必须使用【辅助工具】查看模型列表。
- 【高清植物包】：将**植物**由 128x 分辨率**提高到 512x 分辨率**。请根据您的需求使用！
